a클래스에서는 b라는 클래스의 객체를 직접 만들어서 사용한다면
a-1클래스에서는 b라는 클래스의 객체를 직접 만들어서 사용하는 것이 아니라
외부에서 b라는 클래스의 객체를 만들고 setter와 같은 메서드를 이용해서 b라는 객체를 
사용할 수 있다.

위의 예시가 객체 일체형, 후자가 조립형이고 조립형을 많이 사용해야지 객체 간의 결합력이 
약하기 때문에 효율적이다. 결합력이 약하면 코드의 수정이 편리하고 객체를 손쉽게 교체
할 수 있다.
B b = new B();
A a = new A();
a.setB( b );    //여기서 b라는 객체를 a라는 객체에 주입시킨다.( 의존성 주입 )

이때 위의 예시처럼 setter를 이용해서 주입하는 방법과 생성자를 이용해서 주입하는 방법이
있다.

스프링은 이러한 객체를 외부에서 만들고 개발자가 원하는 객체에 속성값을 주입시켜준다.
즉, DI는 객체를 조립하는 것을 의미한다.
